
# Web Scraping & API Integration Project

## Pershkrimi
Ky projekt projekt realizon web scraping nga nje faqe publike dhe integron nje API per te marre te dhena shtese.
Te dhenat ruhen ne formate CSV dhe JSON.

## Teknologjite e perdorura
- Python
- BeautifulSoup
- Requests
- Pandas
- OpenWeatherMap API
- Google Colab
- Git & GitHub

## Struktura e projektit
- KristiMadolli_web_Scraping_API.ipynb - notebook kryesor
- articles.csv / articles.json - te dhenat nga web scraping
- final_articles.csv / final_data.json - te dhenat e kombinuara me API
- README.md - dokumenti i projektit

## Si ekzekutohet projekti
1. Hap notebook-un ne Google Colab
2. Ekzekuto te gjitha qelite nga lart-poshte
3. Skedaret CSV dhe JSON gjenerohen automatikisht
